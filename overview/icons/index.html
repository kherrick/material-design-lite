Material Symbols

It is possible to use the variable font for Material Symbols with a simple change.

To change an axis of the symbol, use the following variables

:root {
    --material-symbols-fill: 0;
    --material-symbols-wght: 400;
    --material-symbols-grad: 0;
    --material-symbols-opsz: 48;
}
